**🤖 Upcoming AI Workshop — Quick Setup Checklist**

Hey everyone! We're excited to see you for the **upcoming 2-hour hands-on AI workshop**. It's practical, informal, and you'll get real value from your own laptop.

**Before you show up, please check these three things** (takes ~10 minutes):

1️⃣ **VS Code** — Have the latest stable version installed
   → Download here if needed: <https://code.visualstudio.com>

2️⃣ **Copilot Extensions** — Install these two in VS Code:

- GitHub Copilot (the main one)
- GitHub Copilot Chat
   → Just search for "Copilot" in Extensions, click Install

3️⃣ **Verify it works** — Quick test:

- Sign into GitHub from VS Code (bottom left corner)
- Open any `.py` file and start typing a function
- You should see *ghost text* (AI suggestions) appear — that's Copilot working!
- If no ghost text, check GitHub.com → Settings → Copilot to confirm your license is active

**Windows folks**: We recommend installing Windows Terminal (makes everything nicer), but cmd.exe works fine too.

**Nice-to-have**: Have a project or repo open on your laptop to experiment with. If not, no worries—we'll have examples ready.

**⚠️ Can't get Copilot working?** No problem! Come 10 minutes early and we'll troubleshoot together. It's usually a quick fix.

See you at the workshop! 🚀
